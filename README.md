# MultiHUD — CarX Drift Racing Online

A custom HUD mod for **CarX Drift Racing Online**, built with the KSL modding framework.  
Replaces the default speedometer with fully custom, game-inspired instrument clusters featuring a tachometer, speedometer, gear indicator, and turbo boost gauge.

---

## 🎮 Features

- Multiple HUD themes to choose from
- Turbo boost gauge — needle reacts to engine RPM in real time
- Tachometer, digital speedometer, gear indicator
- Automatically switches between KMH and MPH based on your in-game setting
- HUD is only visible during races — hidden in menus and garage
- Each HUD theme remembers its own position and scale
- Everything configurable directly from the KSL mod menu in-game

## 🔧 Installation

1. Download the latest `.zip` from the [Releases](../../releases) page
2. Unpack and place `.ksm` it in your KSL mods folder (or update in KSL):
   ```
   CarX Drift Racing Online\Mods\
   ```
3. Launch the game — the mod will load automatically

---

## 🕹️ How to use

Open the **KSL mod menu** in-game and find **MultiHUD**.

- Press **▶ Open HUDs** to see the main HUD list
- Press **▶ Another HUDs** to see the community HUD list
- Click any HUD name to toggle it on (`[ON]`) or off (`[OFF]`)
- Only one HUD can be active at a time
- Use the **Position X / Y** sliders to move the HUD anywhere on screen
- Use the **Scale** slider to make it bigger or smaller
- Hit **Reset Position** to go back to default
- All settings are saved automatically and loaded next session
